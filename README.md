> Checks if inputted int is a prime number or not

Made with Java 16

## Install

```sh
mvn compile -f "(location of pom.xml)"
mvn package -f "(location of pom.xml)"
```
## Usage

```sh
java -jar .\target\primenumcheck-x.x-SNAPSHOT.jar
```

## Run tests

```sh
mvn test -f "(location of pom.xml)"
```
